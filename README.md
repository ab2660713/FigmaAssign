# Figma Assignment (Dockerized React App)

## 🚀 Project Overview

This is a React + Vite + Tailwind CSS project containerized using Docker.

## 🐳 Docker Image

Docker Hub Link:
https://hub.docker.com/r/ankitbindal6268/figmaassignment

## ▶️ How to Run

### Step 1: Pull Image

```bash
docker pull ankitbindal6268/figmaassignment:latest
```

### Step 2: Run Container

```bash
docker run -p 5173:5173 ankitbindal6268/figmaassignment:latest
```

### Step 3: Open in Browser

http://localhost:5173

## 🛠 Tech Stack

* React
* Vite
* Tailwind CSS
* Docker
